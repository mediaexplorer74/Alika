# Alika
Это мой форк VK-мессенджера [Alika](https://github.com/De-Os/Alika). Штука не для всех / Only for RU Zone.

Заинтересовал проект тем, что там уже есть пред-настроенный VK App ID. То есть мне осталось лишь взять Visual Studio, чуток подправить вылеты аппликашки (что-то там из-за стикеров не взлетало "из коробки"), и ву-аля :)

## Описание 
В функционал входят лишь самые базовые возможности ВК (в основном чатик), но и некоторые особые плюшки, например: [выгрузка диалогов](https://t.me/alika_faq/15) или [просмотр истории редактирования сообщения](https://t.me/alika_faq/14) :)

## Известные проблемы, они же баги
* Могут наблюдаться вылеты на страницах без единого набора стикеров
* Если токен стал недействительным, приложение просто крашится ([решение](https://t.me/alika_faq/11))
* Для полной смены темы нужно полностью перезапустить приложение
* Кривая анимация на некоторых наборах стикеров
* Разрабатывается в свободное время одним человеком (Александром) и предоставляется "как есть".
